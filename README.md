# TikTok-Unlike-All

Tampermonkey userscript that automatically removes likes while browsing any TikTok profile. Manual start/stop control.

---

## Features

- Works only on **profile pages**
- Manual **START / STOP** control
- Detects liked videos and removes the like
- Automatically navigates to the next video
- Runs fully locally in the browser
- Does not send data to any external server

---

## Installation

1. Install the Tampermonkey browser extension  
2. Create a new userscript  
3. Paste and save the script from:

https://raw.githubusercontent.com/tanersb/tiktok-unlike-all/tiktok-unlike-all.user.js


---

## Usage

1. Open any TikTok profile  
2. A **START UNLIKE** button appears on the right side  
3. Click to start  
4. Click again to stop
